# Lume Web Companion

Carregue esta pasta como extensão descompactada em Chrome, Edge ou Brave. Ela envia apenas agente, título sanitizado, origem, estado e um hash local do caminho para `127.0.0.1`; conteúdo da conversa não é lido nem armazenado. Quando solicitado pelo usuário, o Companion também ativa a aba correspondente.
